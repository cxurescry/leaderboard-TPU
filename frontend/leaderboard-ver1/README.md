# Лидерборд ТПУ

Это веб-приложение для отображения рейтинга студентов с интеграцией авторизации через OAuth ТПУ.

## Технологии

- **Frontend**: React 18 + Vite + React Router + Axios + CSS
- **Backend**: FastAPI + SQLAlchemy + SQLite / PostgreSQL + OAuth (ТПУ)
